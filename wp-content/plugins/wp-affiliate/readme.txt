== Description ==

WP-Affiliate is a plugin which allows you to mask affiliate links on the fly from without your Write->Post page.

== Installation ==

1. Upload the files to /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in Wordpress
3. Be Happy
